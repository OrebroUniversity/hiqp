int hiqp::geometric_primitives::GeometricCylinder::init ( const std::vector< double > &  parameters)
inlinevirtual

Parses a set of parameters and initializes the cylinder.

Parameters
parametersshould be of size 8.
Indices 0-2 (required) defines the directional vector of the cylinder,
indices 3-5 (required) defines the position of a point on the cylinder's coaxial line,
index 6 (required) defines the radius of the cylinder,
index 7 (required) defines the height of the cylinder.
Returns
0 on success, -1 if the wrong number of parameters was sent
