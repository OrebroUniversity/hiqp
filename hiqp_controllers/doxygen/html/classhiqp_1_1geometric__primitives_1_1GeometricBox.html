int hiqp::geometric_primitives::GeometricBox::init ( const std::vector< double > &  parameters)
inlinevirtual

Parses a set of parameters and initializes the box.

Parameters
parametersshould be of size 6, 9, or 10.
Indices 0-2 (required) defines the position of the center of the box,
indices 3-5 (required) defines the dimensions of the box,
indices 6-8 (optional) defines euler angles of the orientation of the box (xyz)
indices 6-9 (optional) defines a quaternion for the orientation of the box.
Returns
0 on success, -1 if the wrong number of parameters was sent
